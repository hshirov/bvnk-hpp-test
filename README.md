# HPP Test

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Running Unit Tests

```bash
npm test
```

### Running E2E Tests

Make sure the development server is running:

```bash
npm run dev
```

Run the tests:

```bash
npm run test:e2e
```
